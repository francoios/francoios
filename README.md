## About Me

I'm a CTO & lead dev who builds games and the tools that make building them faster. I created **AnkleBreaker Studio** to combine my two passions: crafting immersive game experiences and developing open-source developer tools.

Right now I'm deep into **MCP servers** — tools that let AI assistants like Claude directly control Unity, Discord, Jira, and other platforms developers use every day.

## What I'm Building

### 🎮 Unity MCP — AI meets game development
My flagship open-source project: **145+ tools** giving AI full control over Unity Editor and Unity Hub.

| | |
|---|---|
| [**unity-mcp-server**](https://github.com/AnkleBreaker-Studio/unity-mcp-server) | Node.js MCP server — the brain connecting AI to Unity |
| [**unity-mcp-plugin**](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) | Unity Editor plugin (UPM) — the bridge inside your editor |

Scene management, GameObject creation, C# scripting, multi-platform builds, animation, physics, lighting, audio, Shader Graph, profiling — all driven by AI.
