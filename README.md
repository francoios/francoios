## 🚀 What I'm Working On

### Unity MCP — AI-Powered Unity Development
My flagship open-source project: a **145+ tool** integration that gives AI assistants like Claude full control over Unity Editor and Unity Hub.

| Repo | Description |
|------|-------------|
| [unity-mcp-server](https://github.com/AnkleBreaker-Studio/unity-mcp-server) | Node.js MCP server — the brain that connects AI to Unity |
| [unity-mcp-plugin](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) | Unity Editor plugin — the bridge running inside your editor |

**What it can do:** Scene management, GameObject creation, component editing, C# scripting, multi-platform builds, animation, physics, lighting, audio, profiling, Shader Graph, and much more — all controlled by AI.
